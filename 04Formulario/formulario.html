Todos los formularios que se manejan en la web utilizan metodos para el envio o respuesta de informacion, los mas comunes son:
GET:Este metodo se encarga de enviar la informacion por la url o salida estandar del navegador, esto propicia que se pueda visualizar toda la informacion del formulario.
POST:Este metodo se encarga de enviar la indormacion por la url pero de forma oculta, lo que ofrece cierto nivel de seguridad, este metodo se usa comunmente para insertar un nuevo registro dentro de los sistemas.
